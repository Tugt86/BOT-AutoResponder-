# Kingdoms AutoResponder Bot

سيرفر ويب لتشغيل بوت AutoResponder للعبة الممالك.

## المميزات:
1. **نظام التسجيل**: تسجيل اللاعبين الجدد مع اختيار لقب
2. **نظام الملف الشخصي**: عرض معلومات اللاعب
3. **نظام الإشعارات**: إشعارات للاعبين عند الحصول على مكافآت
4. **نظام الإنذارات**: اكتشاف اللاعبين في ممالك غير مملكتهم
5. **نظام الدعوة**: مكافأة اللاعبين الذين يدعون لاعبين جدد

## التثبيت على Replit:
1. انشئ مشروع Replit جديد (Node.js)
2. انسخ الملفات إلى المشروع
3. اضغط على زر **Run**

## إعداد AutoResponder:
1. افتح تطبيق AutoResponder for FB Messenger
2. أضف قاعدة جديدة
3. اختر "اتصال بخادم ويب خارجي"
4. أدخل الرابط: `https://your-repl-name.repl.co/webhook`
5. احفظ القاعدة

## بنية البيانات:
يتم حفظ البيانات في مجلد `data/`:
- `players.json`: بيانات اللاعبين
- `pending.json`: عمليات التسجيل قيد الانتظار
- `messages_log.json`: سجل الرسائل (يحتفظ بـ 20 يوم فقط)

## الأوامر المتاحة:
- `تسجيل`: بدء عملية التسجيل
- `ملفي`: عرض الملف الشخصي
